# 🤖 YandexCheckerBot

Бот для проверки номеров телефонов в Яндекс с автоматическим решением капчи.

## 📋 Требования

### ⚠️ Обязательная установка перед запуском:

**1. Tesseract OCR**

⚠️ **Обязательно** установите Tesseract OCR перед использованием бота:

1. Скачайте установщик: **tesseract-ocr-w64-setup-5.5.0.20241111.exe**
   - GitHub: https://github.com/UB-Mannheim/tesseract/wiki
   - Или: https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-5.5.0.20241111.exe

2. Запустите установщик

3. **Важно:** При установке отметьте галочку:
   ```
   ☑ Add Tesseract to the PATH
   ```

4. Рекомендуемый путь установки:
   ```
   C:\Program Files\Tesseract-OCR
   ```

5. После установки **перезапустите терминал/командную строку**

**2. Python 3.13+** (только для разработки)

## 🚀 Быстрый старт

### Вариант 1: Запуск готового .exe (рекомендуется)

1. Убедитесь, что Tesseract OCR установлен
2. Запустите `YandexCheckerBot.exe` или `run.bat`

### Вариант 2: Запуск из исходного кода

```bash
# Установка зависимостей
uv sync

# Запуск бота
uv run python bot.py
```

## 📦 Сборка в .exe

### Автоматическая сборка

```bash
uv run python build.py
```

Скрипт автоматически:
1. ✅ Проверит наличие Tesseract OCR
2. 📦 Установит зависимости
3. 🔨 Соберёт `.exe` файл

### Ручная сборка

```bash
# Установка PyInstaller
uv pip install pyinstaller

# Сборка
pyinstaller --onefile --name YandexCheckerBot --hidden-import easyocr --hidden-import pytesseract --hidden-import telebot bot.py
```

### После сборки

`.exe` файл будет создан в:
- `dist/YandexCheckerBot.exe`
- Копия в корне: `YandexCheckerBot.exe`

### Быстрый запуск через .bat

```bash
# Просто запустите run.bat
run.bat
```

## 📖 Использование

### Команды бота

- `/start` - Приветствие и инструкция
- `/skip` - Пропуск текущей капчи

### Форматы номеров

Отправляйте номера в любом формате:
- `89212810954`
- `+79212810954`
- `+7 921 281-09-54`

### Массовая проверка

**Несколько номеров в одном сообщении:**
```
89212810954
89212810955
89212810956
```

**Ответ:**
```
+7 921 281-09-54: ✅ Аккаунт зарегистрирован
+7 921 281-09-55: ❌ Аккаунт не зарегистрирован
+7 921 281-09-56: ✅ Аккаунт зарегистрирован
```

## 🔧 Решение проблем

### "Tesseract не найден"

1. Убедитесь, что Tesseract установлен
2. Проверьте, что добавлен в PATH
3. Перезапустите терминал
4. Проверьте команду: `tesseract --version`

### Ошибки при сборке

```bash
# Очистите кэш
rmdir /s /q build dist
del *.spec

# Запустите сборку заново
python build.py
```

### Капча не распознаётся

- Бот автоматически повторяет попытки
- Используйте `/skip` для пропуска
- Проверьте качество изображения капчи

## 📝 Зависимости
