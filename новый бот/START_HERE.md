## Что новое?

Ваш Yandex CTF бот теперь доступен в двух формах:


```bash

python bot_new.py

# Терминал 2:
python server.py
```

## 🎯 API примеры

### Проверить один номер (быстро)
```bash
curl -X POST "http://localhost:8000/api/check/phone" \
  -H "Content-Type: application/json" \
  -d '{"phone": "89212810954"}'
```

### Проверить 100 номеров (асинхронно)
```bash
curl -X POST "http://localhost:8000/api/check/batch" \
  -H "Content-Type: application/json" \
  -d '{
    "phones": ["89212810954", "89213456789", ...],
    "max_concurrent": 15
  }'
```

### Смотреть статус
```bash
curl http://localhost:8000/api/status/{task_id}
```

## ❓ Вопросы?

**Q: Нужно ли мне переходить на новую версию?**  
A: Нет, старый `bot.py` по-прежнему работает. Но `bot_new.py` меньше и чище.

**Q: Смогу ли я использовать бот и сервер одновременно?**  
A: Да! Они совместимы и будут обновлять одну статистику.

**Q: Что если я захочу только сервер (без бота)?**  
A: Просто запустите только `python server.py`.

**Q: Как интегрировать с моей системой?**  
A: Импортируйте `checker.py` и используйте функции напрямую, или общайтесь через REST API.
